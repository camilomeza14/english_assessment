# Rock Pesado – Landing Page (HTML & CSS Assessment)

## Description
Landing page for "Rock Pesado", a Colombian's Rock Band that wish promote their idendity, music and next performances around the world

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
project/
├── index.html    → Main HTML file with all sections
├── style.css     → All styles (colors, layout, responsive)
└── README.md     → This file
├── est.jpg       → Photo
├── im2.jpg       → Photo
└── rock.jpg      → Photo
├── sold.jpg      → Photo
└── rockssss.jpg  → Photo
└── kiss.jpg      → Photo


```

## Sections
- **Header** – Rock Band and navigation menu
- **Hero** – Main section with call to action
- **Concerts** – Tasting schedule (HTML table, Extra point)
- **Essence** – General information and Band Rock philosophy
- **Photos** – Image gallery
- **Footer** – Contact info and social media links

## How to View Locally
1. Download or clone the project files.
2. Open `index.html` in any web browser.
3. No additional tools or installations required.

## Responsive Design
- Mobile: max-width 768px — single column layout, compact navigation
- Desktop: min-width 1024px — multi-column grid layout
